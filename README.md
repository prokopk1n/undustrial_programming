# undustrial_programming
Сортировка произведений на английском языке.

Есть два типа запуска.

Сначала команда Makefile.

Далее:
1) ./lab2 test - режим тестирования

2) ./lab2 hamlet.txt file_for_direct_sort.txt file_for_reverse_sort.txt file_for_without_sort.txt 

Где: 
hamlet.txt - исходный текстовый файл на английском языке.

file_for_direct_sort.txt - файл, куда запишется сортировка по началу (должен существовать);

file_for_reverse_sort.txt - файл, куда запишется сортировка по концу (должен существовать);

file_for_without_sort.txt - файл, куда запишется исходный файл (должен существовать).
